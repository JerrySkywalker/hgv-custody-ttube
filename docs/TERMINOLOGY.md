# Terminology

This project uses trajectory tube as the core dissertation concept, corresponding to the Chinese term "轨迹管道".

Pipeline is a software execution-flow term. Use it only for engineering contexts such as DAG steps, cache, manifests, resume behavior, run status, and execution orchestration.

Pipe, trajectory pipe, and tpipe were early temporary engineering names. They are deprecated and should not be used for the repository, MATLAB package, function calls, or the dissertation concept.

Observability Gramian Field is an observability analysis tool or a possible future theoretical submodule under estimation or observability. It is not the engineering project name.
